# Surge — Static Site

Vanilla HTML, CSS, and JS. No build step.

Deploy via Netlify with `publish = "site"` in `netlify.toml`.

## Environment variables (Netlify)

- `SUPABASE_URL`
- `SUPABASE_SERVICE_KEY`
- `ANTHROPIC_API_KEY`
